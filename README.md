# TFG 2026 — Minijuego para Astrapace

## Descripción

Trabajo de Fin de Grado (TFG) consistente en el desarrollo de un minijuego 3D de tipo *endless runner* (al estilo **Subway Surfers**) implementado con **Unity 6.4**, aprovechando el paquete **Splines** para la generación procedural de carriles y obstáculos curvos.

## Tecnologías y versiones

| Tecnología | Versión |
|---|---|
| Unity | 6.4 |
| Unity Splines | 2.8.4 |

## Objetivo

Diseñar e implementar un minijuego endless runner 3D en el que el jugador corre sobre carriles generados mediante splines, esquivando obstáculos y recogiendo objetos, explorando las posibilidades del paquete **Unity Splines 2.8.4** para la creación de caminos curvos y dinámicos.
